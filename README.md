# Slime-Rancher-Assistant
A simple web assistant to help plan out your ranch in the video game Slime Rancher by Monomi Park.

## Run the backend
To run the backend:
`cd api`
`npm run develop`

## Run the frontend
To run the frontend:
`cd frontend`
`npm run dev`
